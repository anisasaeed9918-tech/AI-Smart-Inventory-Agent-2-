# AI Inventory Optimization System

## Project Overview

This project is an **AI-powered Inventory Optimization System** developed using Python and Streamlit.
The system helps businesses analyze sales data, forecast future demand, and suggest optimal pricing strategies using data-driven insights.

The application provides a simple interactive dashboard where users can upload their sales dataset and receive intelligent predictions and pricing recommendations.

---

## Key Features

### Demand Forecasting

The system uses a forecasting model to predict future demand based on historical sales data.
It generates predicted values for the next periods to help businesses plan inventory efficiently.

### Sales Data Analysis

The application reads a CSV file containing sales data and analyzes the trends to understand historical performance.

### Forecast Visualization

A demand forecasting graph is generated to visually represent historical sales data and predicted future demand.

### Dynamic Pricing Suggestion

The system provides a **recommended product price** based on the demand forecast and sales patterns.

## How the System Works

1. The user uploads a **CSV file containing sales data**.
2. The system processes the dataset using Python.
3. A forecasting model predicts demand for upcoming periods.
4. The predicted values are displayed in a table.
5. A demand forecasting graph visualizes the predictions.
6. The system suggests an optimized product price based on demand.
